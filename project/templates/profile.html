How to play!

This game is a mixture of tetris and scrabble, and the objective of the game is to make as many words as possible before the letters reach the top row. You can make any word from 2 letters long to 5 letters long simply by clicking each green tile containing the letter, and each letter you remove will add one point to the total score.

You can see other people's high scores using the leaderboard

Now just click start to start your game!
